# Project Details
## Project Title: Second-Hand Marketplace System
## Problem Statement:
It's a tradition in Albania to hoard things we've rendered useless: clothes that don't fit, old furniture, games that we no longer play due to the "What if we need it later?" mentality. However, that is hardly ever the case. Finding means through which you can gift, sell, or buy things you no longer need are either non-functional, difficult to find, or used incorrectly. 
## Proposed Solution:
We will create an online platform, locked to Albanian citizens, where users are able to list, discover, purchase, and sell clothes and other items to other items. Our platform's features (which include but are not limited to searching, messaging, rating, and reporting) will ensure that the final product we offer is a usable, safe software that Albanian citizens will find very useful.
## Project Scope:
- Aim:To create a secure and easy to use digital marketplace that encourages sustainability and decluttering for users inside Albania.
- Objectives:
1. Building a system that ensures safety for both sellers and buyers.
2. Reducing customer waste and promoting second hand shopping.
3. Search features and filters that make it possible for users to find what they need instantaneously. 
## Application Description:
The application makes it possible for buyers and sellers to have a safe marketplace to exchange second hand items. Users create an account and verify their identity. Afterwards they're free to buy and sell. Sellers can list items by providing photos or videos, setting a price adding description and tags. Buyers can search for items and filter out what they're looking for. Users can message each other to ask further questions for the product and also have an option to negotiate prices. They will have different delivery options from which they can choose from.
